# ofdb-seed

A [seed](https://seed-rs.org) library for building
[Open Fair DB](https://github.com/kartevonmorgen/openfairdb)
based web applications.

## Usage

```toml
[dependencies]
ofdb-seed = "*"

[patch.crates-io.ofdb-seed]
git = "https://github.com/kartevonmorgen/ofdb-seed"
```
